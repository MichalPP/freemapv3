GpsVP


GpsVP je open source program pre prácu s vektorovými Garmin a online rastrovými mapami (po stiahnutí máp do cache umožnuje aj offline režim). Program beží pod Windowsami na desktope a aj na PDA.

Zobrazenie freemap.sk v GpsVP


  • stiahnite program a nainštalujte ho (vo Windows XP, Vista a pod. stačí rozbaliť)
  • vytvorte adresár MapConfigs v adresári, kde bol GpsVP nainštalovaný (rozbalený)
  • vytvorte súbor Freemap.ini v adresári MapConfigs s obsahom
    [Tiled MAP]gpsVPVersionMin=0.4.15Filename=x=%X&y=%Y.pngURL=http://www.freemap.sk/layers/allinone/?/%ZOOM_00/%X/%Y

  • nastavte v menu -> Maps -> Raster Maps -> Set raster cache folder nejaký adresár na disku. Do tohto adresára bude program ukladať stiahnuté obrázky, takže je dôležité, aby tam bol dostatok miesta
  • v menu -> Maps -> Raster Maps -> Raster Map Type by sa mal objaviť Freemap, vyberte ho
  • zapnite v menu -> Maps -> Raster Maps -> Enable raster downloading
  • ak si želáte lepšie riadiť, čo mám byť na mape - url môžete vydekorovať o písmenka podľa LayerAllInOne - čím môžete získať napr. mapu s vrstevnicami a/alebo turistickými chodníkmi a pod.
